4 版本安装升级操作指导 操作前确认:软件升级或重新安装前,请确认服务器的网络参数(IP 地址、子网掩码、网关)配置。 服务器与客户端计算机的网络连接正常。4.1 软件安装(1)将 IA-VD Server.exe 拷贝到服务器上,双击。(2)弹出安装向导窗口,点击<下一步>。 IA-VD Server 安装向导    (3)选择安装文件夹,点击<下一步>,选择安装文件夹 5
 (4) 点击<安装>。 安装 IA-VD Server(5)弹出 IA-VD Server 配置工具窗口,输入视频诊断服务器 IP 地址、VM 服务器地址、VM 数据 库地址、VM 用户名密码,默认勾选乱序整理和、抗丢包,点击<确认>。输入参数说明:视频诊断服务器 IP 地址:填写 IA-VD SERVER 安装的服务器的地址,即 IA-VD SERVER 安装的 本地地址VM 服务器地址:填写 VM 的地址VM 数据库地址:填写 VM 服务器的地址VM 用户名:VM 的用户,一般为 loadmin VM 密码:VM 用户对应的密码  6
IA-VD Server 配置工具 (6)点击<完成>,结束 IA-VD Server 安装。 完成 IA-VD Server 安装  7
